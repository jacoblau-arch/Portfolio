This repository contains hands-on networking and security projects built using Cisco Packet Tracer and other infrastructure tooling.

## 1. Secure Multi-VLAN Small Business Network

A segmented multi-VLAN small office network design using a Cisco 2911 router and 3560 multilayer switch.

Includes:
- Inter-VLAN routing using SVIs
- DHCP relay from a centralized server
- NAT overload at the router edge
- ACL-based segmentation (Admin, Office, Server, IoT, Guest, Management)
- Cisco device hardening for secure management
